# Trading

## Cara menjalankan

Buka terminal (split atau buka new window) sebanyak yang dibutuhkan

```bash
go run ./main.go # window 1 (untuk melihat order yang sedang dieksekusi)
go run ./orderBook.go # window 2 (untuk melihat book order)
go run ./cekSaldo.go # window 3(untuk melihat margin dari wallet)
```
